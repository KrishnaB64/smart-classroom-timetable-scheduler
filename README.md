# Smart Classroom and Timetable Scheduler (SCTS)

## Overview

The Smart Classroom and Timetable Scheduler (SCTS) is a web-based academic management system developed to simplify academic operations within educational institutions. The system provides a centralized platform for managing attendance, timetable scheduling, assignment handling, and academic communication for Students, Faculty, and Admin users.

---

## Features

* Student Attendance Management
* Timetable Scheduling System
* Assignment Upload and Submission
* Academic Notifications
* Role-Based Authentication
* Admin Dashboard
* Responsive User Interface

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/smart-classroom-timetable-scheduler.git
cd smart-classroom-timetable-scheduler
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Apply migrations:

```bash
python manage.py migrate
```

Run the server:

```bash
python manage.py runserver
```
